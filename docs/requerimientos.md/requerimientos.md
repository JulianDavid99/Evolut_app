# Requerimientos del Sistema – Evolut

## 1. Descripción general

Evolut es una aplicación web diseñada para ayudar a las personas a reflexionar sobre decisiones, hábitos y comportamientos relacionados con diferentes áreas de su vida.

El sistema presenta categorías de reflexión donde el usuario puede identificar una situación personal, leer reflexiones orientadoras y escribir su propia reflexión.

Las reflexiones del usuario son privadas y solo pueden ser vistas por el administrador del sistema, quien puede responder con retroalimentación personalizada.

---

## 2. Tipos de usuario

### Usuario
Persona que utiliza la aplicación para reflexionar sobre situaciones personales.

Funciones del usuario:
- Registrarse en el sistema
- Iniciar sesión
- Seleccionar una categoría
- Leer reflexiones orientadoras
- Escribir una reflexión personal
- Ver el historial de sus reflexiones
- Leer respuestas del administrador

### Administrador
Persona encargada de revisar y responder reflexiones de los usuarios.

Funciones del administrador:
- Iniciar sesión
- Ver reflexiones enviadas por los usuarios
- Responder reflexiones
- Ver la categoría asociada a cada reflexión

## 3. Funcionalidades del sistema

El sistema debe permitir:

- Registro de usuarios
- Inicio de sesión
- Visualización de categorías
- Visualización de reflexiones orientadoras
- Registro de reflexiones personales
- Historial de reflexiones del usuario
- Panel de administrador para revisar reflexiones
- Respuesta del administrador a reflexiones

## 4. Reglas del sistema

- Cada usuario solo puede ver sus propias reflexiones.
- Las reflexiones son privadas.
- Cada reflexión pertenece a una categoría.
- El administrador puede responder a cualquier reflexión.
- Los usuarios pueden ver las respuestas del administrador en su historial.

## 5. Categorías iniciales del sistema

El sistema comenzará con las siguientes categorías:

- Decisiones
- Disciplina
- Productividad
- Responsabilidad personal

En futuras versiones podrán agregarse más categorías.
