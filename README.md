# Evolut

Aplicación web diseñada para ofrecer claridad directa y confrontativa a adultos que atraviesan momentos de confusión o malas decisiones.

## Objetivo

Construir el proyecto siguiendo el ciclo completo de vida del software:
- Documentación formal
- Uso profesional de Git
- Organización empresarial
- Arquitectura clara
- Evidencia para portafolio técnico
